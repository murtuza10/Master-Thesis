```json
{
  "Crops": [
    {"cropSpecies": { "value": "apple", "span": [84, 89] }}
  ],
  "Soil": [],
  "Location": [
    {"country": { "value": "Germany", "span": [147, 152] }},

    {"endTime": { "value": "2003", "span": [176, 180] }}

```
Note: The "span" value for "cropSpecies" is the start and end character positions from the beginning of the full text, which in this case is the string "apple" at positions 84 to 89. Similarly, the "span" value for "country" is the start and end character positions for the string "Germany" at positions 147 to 152.
